# Brand sources

Copy logo PNGs from `doc/logos/` here (or place them directly in `doc/logos/`):

- `logo-square.png` — header mark
- `logo.png` — favicon / apple-touch source
- `logo-horizontal.png` — footer / OG composite
- `logo-squre-text.png` — square + wordmark (optional)

Run `pnpm assets:brand` to generate optimized files under `public/brand/`, `public/favicon.ico`, and `public/og/og-default.jpg`.
